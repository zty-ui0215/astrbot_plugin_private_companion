# 更新日志

## 3.0.1

- 拓展页总览新增“拓展页自检”面板，集中提示角色设定、私聊目标、群聊入口、图片转述缓存、功能开关密度和 AI 早报文字版读取状态。
- 优化拓展页排版断点与角色设定页布局，改善中等宽度、移动端、热力图、模块摘要和用户区域的挤压问题。
- 角色设定页将外貌信息中的“外貌”调整为“识别点”，并保留旧字段回填兼容。
- 用户设定补充性别、年龄、职业字段，标准化填写、示例填入和旧版文案保存保持一致。

## 3.0.0

- 拓展页新增独立“角色设定”页，支持旧版文案与标准化填写切换，按角色、世界观、用户三块维护设定。
- 角色设定补充外貌信息分组，图片自我识别仅注入已填写的年龄、性别、外貌、发型发色、瞳色和服饰风格等有效字段。
- 合并转发消息适配增强，支持读取消息集文字、解析节点结构，并可对合并消息内图片生成视觉摘要。
- AI 早报追踪增强 B 站投稿接口探测、文字版读取和调试状态输出。

## 2.9.0

- 新增“图片转述增强”功能开关，整合私聊单图防抖、视觉转述等待和 Bot 自我识别线索。
- 私聊单图回复链路改为等待视觉转述完成后再进入主链，并补充空回复兜底，避免只发图时无回复。
- 视觉摘要新增“图像表达意图”和“图像归属判断”，自我识别仅作为补充线索，优先理解用户发图想表达的情绪、文字或梗。
- 重复图片/表情包按内容哈希缓存视觉摘要，避免同图反复调用识图模型；用户负面反馈会使相关缓存失效并重新识别。
- 拓展页“图片转述增强”详情支持调整等待秒数、重复图片转述缓存和自我识别线索。

## 2.8.1

- 新增“阻止群聊现实承诺”开关，仅影响群聊；开启后 Bot 不会承诺自己能拉人、修网、开房间或操作现实设备，私聊扮演不受限制。
- 优化群聊唤醒判定，`@` 其他群友、回复其他群友或 `@全体` 不再被误判为正在叫 Bot，兴趣唤醒也会跳过这些场景。
- 主动分段补强括号、双引号和网址保护，避免链接或括号内内容被拆成不自然的多条消息。
- 书柜漫画阅读页的 Bot 批注改为浮层/折叠展示，避免挤压图片影响阅读。
- 调整主动分段默认值：最多 3 段，不分段阈值默认 500 字，减少主动消息刷屏感。

## 2.8.0

- 新闻阅读与主动搜索加入“外界信息自我关联”链路：Bot 会先判断外界消息和自己的模型、能力、兴趣、创作、日程或关系有什么关系，再决定是否形成主动分享欲。
- AI 早报类 B 站消息源增强：支持 UP 主源与单视频源，优先读取视频简介里的文字版链接，并尽量以完整正文作为新闻整理依据。
- B 站消息源优先复用已加载的 B 站插件/BiliBot 客户端，公开接口不可用时再回退搜索路径。
- 主动消息分段加入智能网址保护，避免 `https://...` 被句号、斜杠或分段词拆成多条消息。
- 拓展页功能开关新增外界信息自我关联配置，可调整分享欲倍率和冷却时间，并在新闻阅读、主动搜索二级页联动展示。
- 新闻与搜索分享上下文补充“和自己有关的地方、表达气质、额外边界”，让最终表达继续由人格生成，而不是变成固定播报。

## 2.7.0

- 新增外部插件接入 API，其他插件可以把自己的能力注册为本插件的外部主动能力。
- 外部主动能力支持拓展页管理，可配置启用状态、触发权重、冷却时间和自定义 JSON 参数。
- 主动行为编排可识别 `external:<能力名>`，并把外部能力纳入日程、心情、动机和主动候选决策。
- README 增加外部插件接入示例，便于第三方插件把主动行为有机接入陪伴体系。
- 补齐外部能力的持久化、执行记录、失败状态和插件卸载时的 API 引用释放。

## 2.6.0

- 拓展页“功能开关”二级页面支持直接编辑关联参数，并为参数补充中文说明、输入控件和保存反馈。
- 拟人身体状态、被动状态注入、生理周期模拟已接入功能开关体系，可在二级页调整拟人状态强度。
- 浏览记录从书柜夹层迁出，改为在主动搜索记录中以时间线展示，并保留新闻阅读与主动搜索历史。
- 日记、本子读后感和页边批注补强 AstrBot 默认人格、生活人设、当前状态与世界观注入，减少插件模板腔。
- 主动搜索模型命名调整为“主动搜索决策/整理模型”，明确联网检索仍走 AstrBot 全局网页搜索配置。
- 优化私聊单图防抖与视觉转述流程，主模型具备视觉能力时走普通图片消息链，否则并行转述后等待用户补充。
- 书柜本子详情区分简介与 Bot 读后感，避免同一段文本重复展示。
- 补齐无聊看新闻、空档自主搜索等子能力在拓展页中的开关与参数管理。

## 2.5.0

- 拆分“新闻阅读”和“主动搜索”：新闻阅读每日随日程生成获取热点与 RSS 见闻；主动搜索按 Bot 的兴趣、日程和心情自行检索。
- 参考热点监控思路，为新闻阅读加入公开热点候选聚合，并与新闻源一起整理成今日见闻。
- 主动搜索记录会留下搜索词、动机、摘要和来源，并可作为后续状态、日记与主动消息素材。
- 书柜夹层新增“浏览记录”书册，像日记本一样收纳新闻阅读和主动搜索留痕。
- 拓展页总览新增“今日新闻见闻”和“主动搜索记录”观察卡片。
- 修复保存群聊观察后，合并消息阅读的模式、条数、字数上限和视觉图片上限回填为空的问题。
